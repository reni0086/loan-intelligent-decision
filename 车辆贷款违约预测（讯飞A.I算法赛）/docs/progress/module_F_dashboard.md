# 模块 F：可视化看板

## 模块目标
提供可运行的前端看板页面，展示关键业务与风险指标，并与后端 API 联动。

## 已完成功能
- 搭建本地看板页面：
  - `dashboard/index.html`
  - `dashboard/main.js`
- 卡片指标：
  - 实时事件数
  - 实时决策数
  - 系统健康状态
- 图表展示（ECharts）：
  - 风险等级分布饼图
  - 风险趋势折线图
- API 联动：
  - 周期拉取 `/health` 与 `/stats/overview`。

## 实现方式
- 静态页面由 Flask 提供：
  - `/` 返回看板主页
  - `/dashboard/<filename>` 返回前端资源

## 验证结果
- 页面资源完整。
- 与后端健康和统计接口可联动读取数据。

## 未完成项
- 地图热力图与 SHAP 可解释图深度可视化（当前为基础看板）。

## 风险与下一步
- 风险：离线示例图表与真实业务图可能存在语义偏差。
- 下一步：进入模块 G，整理完整交付文档与验收清单。
