## 七、数据修复功能演示

### 7.1 FP-Growth 关联规则修复

FP-Growth 算法挖掘类别特征之间的关联规则，修复缺失的工作类型、地区、信用等级等字段。

**示例场景**：某客户 `employment_type`（工作类型）缺失，但 `credit_score=580` 且 `disbursed_amount=15000`

**FP-Growth 修复逻辑**：
```
规则示例：
IF credit_score=580 AND disbursed_amount=15000
THEN employment_type=1 (置信度 82%)

→ 修复结果：employment_type=1，置信度 0.82
```

### 7.2 ALS 矩阵分解修复

ALS（交替最小二乘法）通过矩阵分解补全数值型缺失特征，如信用评分、贷款金额、月供等。

**修复效果**：
- 覆盖率：90%+ 的缺失值被修复
